## Core Concepts and Theory

### 1. Introduction to Programming Languages

Programming languages are used to give instructions to a computer. In the beginning, programs were written using machine language, which contained only binary values such as `0` and `1`.

Later, assembly language was introduced, which used short commands like `ADD`, `MOV`, and `SUB`. After that, high-level languages such as Fortran, COBOL, and C made programming easier and more understandable.

C is a procedural programming language, while C++ is an extension of C that supports both procedural and object-oriented programming.

---

### 2. Procedural Programming

Procedural programming is a programming approach where the program is divided into different functions. Each function performs a specific task.

In C programming, data and functions are written separately. The data is passed to functions whenever needed.

#### Features of Procedural Programming

- Program is divided into functions.
- Data and functions are separate.
- Execution is done step by step.
- It is simple and fast for small programs.
- It becomes difficult to manage in large programs.
- Data security is low because data is not hidden properly.

---

### 3. Object-Oriented Programming

Object-Oriented Programming is a programming approach based on classes and objects. A class acts like a blueprint, while an object is a real instance of that class.

In C++, data and functions can be combined together inside a class. This makes the program easier to manage, reuse, and modify.

#### Main Features of OOP

- **Class:** A blueprint used to create objects.
- **Object:** An instance of a class.
- **Encapsulation:** Wrapping data and functions together.
- **Abstraction:** Showing only important details and hiding unnecessary details.
- **Inheritance:** Reusing properties of one class in another class.
- **Polymorphism:** Ability of the same function to behave differently in different situations.

---

## Difference Between C and C++

| Basis | C Programming | C++ Programming |
|---|---|---|
| Programming Approach | Procedural | Procedural + Object-Oriented |
| Header File | `#include <stdio.h>` | `#include <iostream>` |
| Input Statement | `scanf()` | `cin` |
| Output Statement | `printf()` | `cout` |
| Format Specifier | Required | Not required |
| Class and Object | Not supported | Supported |
| Data Security | Less secure | More secure |
| Code Reusability | Limited | High |
| Program Management | Difficult for large programs | Easier for large programs |
