## Lab Contents

1. **Daily Sales Tracker (`sales.cpp`)**
   * Stores daily sales for a week in an array.
   * Calculates and displays total and average sales.

2. **Patient Records Structure (`patient.cpp`)**
   * Uses an array of structures (`Patient`) to store records for 3 patients.
   * Fields include: name, age, and disease.

3. **Currency Converter (`currency.cpp`)**
   * Implements a `convertCurrency` function to convert currency amounts based on a user-provided rate.

4. **Bill Calculation (`bill.cpp`)**
   * Passes an `Item` structure into a `calculateBill` function to compute a 10% tax addition.

5. **Movie Theater Seat Management (`theater.cpp`)**
   * Uses a 10-element binary array to track, display, and update seat bookings.

6. **Low Stock Filter (`product.cpp`)**
   * Tracks an inventory of 5 products via a structure array and filters items with a quantity under 10.

7. **Expense Ledger (`expense.cpp`)**
   * Utilizes reference arrays and structures to add items and compute total costs by category.
